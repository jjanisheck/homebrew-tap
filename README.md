# Homebrew Tap

Homebrew cask for [Vectorito](https://github.com/jjanisheck/vectorito) — a macOS
menu-bar app that turns images into clean SVG with AI-picked settings.

```bash
brew install --cask jjanisheck/tap/vectorito
```

Vectorito is unsigned; if Gatekeeper blocks it, right-click the app in
`/Applications` and choose **Open** (or run
`xattr -dr com.apple.quarantine /Applications/Vectorito.app`).
